# Bible App

Website written in [React](https://github.com/facebook/react) using [Create React App](https://github.com/facebook/create-react-app)

## Live version

https://biblia.kosciolwroclaw.pl

## Dependencies

Package | Version
--- | ---
react | ^16.13.0
react-dom | ^16.13.0
react-scripts | ^3.4.0

## Installation

* `npm install`

### Development

* `npm start`

### Production

* `npm run build`
